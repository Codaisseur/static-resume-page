### ðŸŒŸ CSS (Cascading Style Sheets) is a stylesheet language that describes the presentation of an HTML document, defining how elements must be rendered on screen, on paper, or in other media.

With CSS, you can control a lot of things related to how your page looks like:

+ color of the text
+ style of fonts
+ spacing between paragraphs
+ how columns are sized and laid out
+ what background images or colors are used
+ layout designs
+ variations in display for different devices and screen sizes
+ ...
